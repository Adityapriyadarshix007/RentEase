# RentEase Frontend

This is the frontend application for RentEase - Furniture & Appliance Rental Platform.

## Tech Stack

- React 18
- Tailwind CSS for styling
- React Router v6 for navigation
- Axios for API calls
- React Hot Toast for notifications
- React Icons for icons
- Chart.js for analytics

## Getting Started

### Prerequisites

- Node.js 14+
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
